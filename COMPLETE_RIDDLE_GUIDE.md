## ğŸ” Encryption Methods Reference

### 1. Caesar Cipher
- **How it works:** Shifts each letter by a fixed number of positions
- **Example:** A â†’ H (shift +7), "HELLO" â†’ "OLSSV"
- **Decryption:** Shift back by the same amount
- **Tools:** CyberChef, dcode.fr, or manual calculation

### 2. ROT13
- **How it works:** Special case of Caesar Cipher (shift +13)
- **Example:** "HELLO" â†’ "URYYB"
- **Unique feature:** Applying ROT13 twice gives the original text
- **Tools:** rot13.com, CyberChef

### 3. Morse Code
- **How it works:** Dots (.) and dashes (-) represent letters
- **Example:** "HELP" â†’ ".... . .-.. .--."
- **Spacing:** Letters separated by spaces, words by `/` or multiple spaces
- **Tools:** morsecode.world, CyberChef

### 4. Binary (8-bit ASCII)
- **How it works:** Each character as 8-bit binary number
- **Example:** "A" â†’ `01000001`
- **Format:** Usually space-separated bytes
- **Tools:** rapidtables.com, CyberChef

### 5. Hexadecimal
- **How it works:** ASCII values in hexadecimal (base-16)
- **Example:** "HELLO" â†’ `48 45 4C 4C 4F`
- **Format:** Space or colon-separated pairs
- **Tools:** asciitohex.com, CyberChef

### 6. Base64

- **Identification:** Often ends with `=` or `==` padding
- **Tools:** base64decode.org, CyberChef
